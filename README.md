# To-Do List

## 📋 Descrição

Aplicação de lista de tarefas que salva, exibe, edita e remove tarefas, com integração ao back-end e banco de dados.

## 🛠️ Tecnologias

- NodeJs
- MongoDB
- ExpressJs
- ReactJs
